# Kubernetes Nginx DevOps Deployment

Este projeto demonstra a criação e orquestração de uma aplicação Nginx utilizando Kubernetes.

## Tecnologias
- Docker
- Kubernetes
- Nginx
- ConfigMap
- Secret
- Volume (emptyDir)

## Funcionalidades
- Deployment com múltiplas réplicas
- Configurações externas via ConfigMap
- Dados sensíveis via Secret
- Volume para persistência temporária
- Exposição via Service (NodePort)

## Como executar
1. Criar ConfigMap e Secret
2. Aplicar Deployment
3. Expor a aplicação com Service
4. Acessar via navegador
